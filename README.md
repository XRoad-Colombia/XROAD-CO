# X-Road 7.0.4 Versión Colombia

Repositorio con los paquetes de instalación del Servidor de Seguridad de X-Road 7.0.4 en la versión NIIS.

Este release está basado en la versión [7.0.4](https://github.com/nordic-institute/X-Road/releases/tag/7.0.4) del [NIIS](https://www.niis.org/):

- [x] Esta versión se utiliza únicamente para procesos de transición y migración de X-Road Colombia 6.25 a versiones superiores como 7.2.2 o 7.3.2.

A continuación se listan los enlaces de descarga de los paquetes de instalación por sistema operativo:

* [Ubuntu 18.04](https://github.com/XRoad-Colombia/XROAD-CO/releases/download/7.0.4/ubuntu20_7.0.4.zip)
* [Red Hat 7](https://github.com/XRoad-Colombia/XROAD-CO/releases/download/7.0.4/rhel7_7.0.4.zip)
* [Red Hat 8](https://github.com/XRoad-Colombia/XROAD-CO/releases/download/7.0.4/rhel8_7.0.4.zip)
